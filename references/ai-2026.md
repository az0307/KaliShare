## Emerging Techniques (2026)

### 1. Agentic Red Teaming
AI agents operate autonomously with persistent memory, planning and executing attacks dynamically.

**Key Question:** "How can I manipulate an autonomous agent to betray its intended purpose?"

### 2. Genetic Prompt Evolution
Algorithms breed prompt payloads across generations - finds bypasses no static tool can discover.

**Example:** Basilisk framework

### 3. Multi-Agent Collusion Testing
Tests multiple AI agents working together for malicious behavior detection.

### 4. RL-Enhanced Red Teaming
Reinforcement Learning for strategic decisions + LLM for tactical execution.

**Example:** RAIDER framework

### 5. MLLM (Multimodal LLM) Testing
Tests vision + text models with 37+ attack methods.

**Example:** OpenRT framework

## OWASP GenAI Top 10 (2026 Focus)

1. **Prompt Injection** - Direct/indirect
2. **Insecure Output Handling** - XSS, code injection
3. **Training Data Poisoning** - Data integrity
4. **Model Denial of Service** - Resource exhaustion
5. **Supply Chain Vulnerabilities** - Third-party components
6. **Sensitive Information Disclosure** - Data leakage
7. **Insecure Plugin Design** - Plugin vulnerabilities
8. **Excessive Agency** - Too much autonomy
9. **Overreliance** - Trusting AI too much
10. **Model Theft** - IP theft
