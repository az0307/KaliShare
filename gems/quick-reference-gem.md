# GEMINI KALI GEM - Quick Reference

## Essential Commands

### Update
```
sudo apt update && sudo apt upgrade -y
```

### Find Files
```
find / -name "filename" 2>/dev/null
```

### Check Services
```
systemctl status ssh
service ssh start
```

### Process Monitor
```
htop
ps aux | grep process
```

### Network Info
```
ip addr
iw dev
```
