## ⚡ QUICK INSTALL SCRIPT

```bash
#!/bin/bash
# Install ALL the things!

# Create tools directory
mkdir -p /opt/github-tools
cd /opt/github-tools

echo "Installing popular tools..."

# AI Tools
git clone --depth 1 https://github.com/0x4m4/hexstrike-ai.git

# Recon
git clone --depth 1 https://github.com/smicallef/spiderfoot.git
git clone --depth 1 https://github.com/s0md3v/ReconDog.git
git clone --depth 1 https://github.com/projectdiscovery/subfinder.git

# Web
git clone --depth 1 https://github.com/sqlmapproject/sqlmap.git
git clone --depth 1 https://github.com/s0md3v/XSStrike.git
git clone --depth 1 https://github.com/ticarpi/jwt_tool.git

# Exploit
git clone --depth 1 https://github.com/swisskyrepo/PayloadsAllTheThings.git

# Privilege Escalation
curl -L https://github.com/carlospolop/PEASS-ng/releases/latest/download/linpeas.sh -o linpeas.sh
chmod +x linpeas.sh

# Wordlists
git clone --depth 1 https://github.com/danielmiessler/SecLists.git
git clone --depth 1 https://github.com/fuzzdb-project/fuzzdb.git

echo "✅ All tools installed!"
ls -la
```
