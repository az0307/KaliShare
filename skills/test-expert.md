# Test Expert

## Quick Commands

### Python Testing
```bash
pytest                    # Run all tests
pytest file.py           # Run specific file
pytest -k "test_name"    # Run by name
pytest --cov=src         # With coverage
pytest -v                # Verbose
```

### JavaScript Testing
```bash
npm test                 # Run tests
jest --coverage          # With coverage
jest --watch            # Watch mode
```

### Other
```bash
python -m unittest discover
pytest --fixtures       # List fixtures
```
